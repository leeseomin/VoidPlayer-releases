## Keyboard Shortcuts

| Shortcut | Action |
| --- | --- |
| <kbd>Command</kbd> + <kbd>3</kbd> | Cycle background color |
| <kbd>Command</kbd> + <kbd>8</kbd> | Switch to next UI mode |
